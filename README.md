# 🚀 Apex Coding Challenges

This repository contains a curated collection of **Salesforce Apex coding challenges** designed for interview preparation and real-world scenario practice.

---

## 📌 About This Repository

The goal of this project is to:

* Strengthen Apex programming skills
* Practice real-world Salesforce scenarios
* Prepare for Salesforce Developer interviews
* Showcase problem-solving and clean coding practices

---

## 🧠 Topics Covered

* Apex Programming Fundamentals
* Triggers & Trigger Frameworks
* SOQL & SOSL Optimization
* Governor Limits Handling
* Collections (List, Set, Map)
* Batch Apex & Async Processing
* Integration (REST/SOAP Callouts)

---

## 📂 Repository Structure

```
apex-coding-challenges/
```
